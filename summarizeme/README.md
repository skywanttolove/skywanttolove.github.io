# 🌸 สรุปให้ · SummarizeMe

เว็บช่วยสรุปเลกเชอร์ / YouTube / PDF ด้วย AI ที่รันฟรีบนเครื่องตัวเอง (Ollama)

**เปิดใช้:** https://skywanttolove.github.io/summarizeme/

หน้าเว็บโฮสต์ฟรีบน GitHub Pages ได้ แต่ตัว AI ประมวลผลบนเครื่องคุณเอง (ฟรี ไม่ต้องจ่ายค่า API) จึงต้องรัน backend ค้างไว้ตอนใช้งาน

---

## เริ่มใช้งาน (ครั้งแรก)

**1) ลง Ollama + โหลดโมเดล**
โหลดจาก https://ollama.com/download แล้วเปิดโปรแกรม จากนั้นใน Terminal/PowerShell:
```
ollama pull qwen2.5:7b
```
เครื่องแรมน้อย (8GB): ollama pull llama3.2:3b · qwen2.5:7b = ค่าเริ่มต้น (เก่งภาษาไทย/คณิต ต้องการ RAM ~8GB+)

**2) รัน backend** (โฟลเดอร์ `backend/`) — ต้องมี Node.js 18+
```
cd backend
npm install
npm start
```
ขึ้น `🌸 SummarizeMe backend ▶ http://localhost:8787` = พร้อม (เปิดค้างไว้ อย่าปิด)

**3) เชื่อมหน้าเว็บ**
เปิดเว็บ → กดเฟือง ⚙ → ใส่ `http://localhost:8787` → "ทดสอบการเชื่อมต่อ" ให้ขึ้นเขียว → บันทึก (ครั้งเดียว)

---

## จุดเด่นของตัวสรุป (เวอร์ชันปรับแต่งแล้ว)
- **อ่านเนื้อหายาวได้จริง** — ตั้ง `num_ctx` กว้าง + ระบบ map-reduce ย่อเนื้อหายาวเป็นก้อนก่อนสรุป (ดีฟอลต์ Ollama เห็นแค่ 2048 token จะสรุปห้วน)
- **แยกสร้างเนื้อหากับข้อสอบคนละรอบ** — ได้ blocks ครบและ quiz 3 ข้อชัวร์ (โมเดล 8B ทำพร้อมกันมักได้ quiz ไม่ครบ)
- **JSON ทนทาน** — ซ่อมวงเล็บที่ขาด/รูปแบบเพี้ยนจากโมเดลเล็กได้
- **รองรับ https → localhost** (Private Network Access) เปิดจาก GitHub Pages แล้วต่อ backend ในเครื่องได้

## ปรับแต่งเพิ่มได้ (env)
```
MODEL=qwen2.5:7b NUM_CTX=8192 PORT=8787 npm start
```

## ปัญหาที่พบบ่อย
- **ทดสอบแล้วแดง** → ยังไม่เปิด Ollama/backend หรือ URL ผิด
- **YouTube ดึงซับไม่ได้** → คลิปปิดคำบรรยาย ให้ก็อปข้อความมาวางแท็บ "เลกเชอร์" แทน
- **สรุปช้า** → โมเดลใหญ่/เครื่องไม่แรง ลอง `llama3.2:3b`
